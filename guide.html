  1. 이용약관 동의
    프로그램 최초 실행 시 표시. 내용 확인 후 동의.
  2. 이름 입력
    GOE 메신저에 등록된 본인 이름을 입력합니다.
  3. GOE 메신저 아이디 확인
    백업 또는 DB에서 감지된 아이디를 순서대로 보여줍니다.
    본인 아이디가 표시되면 y, 아니면 n을 입력하세요.
  4. PIN 4자리 설정 → PIN 확인 (재입력)
    * 설정 직후 PIN을 한 번 더 요구합니다.
    정상 동작이며, 방금 설정한 PIN을 입력하면 됩니다.
  5. 데이터 소스 선택
    라이브 DB(@Talk.db) 또는 백업 파일(.bak)을 선택합니다. 위 화면 참고.
  6. 첨부파일 폴더 지정 (선택)
    GOE 메신저 다운로드 폴더를 자동으로 찾습니다. 여러 경로 추가 가능.
  7. 처리 완료 후 브라우저로 열림
    * 매 열람 시 반드시 EXE를 다시 실행하십시오.
    보안을 위해 임시 복호화 파일이 생성되어 브라우저에서 열립니다.
    브라우저 북마크로 직접 접근하지 마시고, 항상 EXE를 통해 실행하세요.

일반 백업 경로 (참고)

C:\Users\사용자명\문서\GOE메신저\

5. 주요 기능

쪽지 인덱스 (연도별)

쪽지는 연도별로 분류됩니다. 연도 선택 → 대상자 목록 → 상세 보기. 탭으로 받은쪽지/보낸쪽지/전체를 필터링할 수 있습니다.

보낸 쪽지 "외 N명"

여러 명에게 동시 발송한 쪽지는 발신자 옆에 외 25명 뱃지가 표시됩니다. 받는이를 클릭하면 전체 수신자 확인.

즐겨찾기 (별 아이콘)

별(★)을 클릭하면 즐겨찾기에 등록됩니다. 등록된 별은 노란색으로 표시.

  • 대상자 제목 옆 - 페이지 상단 제목(h1) 우측의 별. 해당 대상자 전체를 즐겨찾기.
  • 쪽지 메시지 카드 - 각 메시지 발신자/시각 행 우측의 별. 개별 쪽지 즐겨찾기.
  • 대화 메시지 버블 - 각 채팅 버블 발신자 옆의 별. 개별 대화 즐겨찾기.
  • 사이드바 별 아이콘 - 클릭 시 즐겨찾기 패널이 열려 등록 목록 확인.

즐겨찾기는 브라우저 로컬 저장소(localStorage)에 보관됩니다. 데이터를 재변환하면 초기화됩니다.

인쇄

브라우저의 인쇄 기능(Ctrl+P)으로 인쇄 친화 레이아웃으로 출력됩니다. 검색/필터/내비게이션은 인쇄 시 자동 숨김.

6. FAQ

라이브 DB는 어디에 있나요?
GOE 메신저가 설치된 PC에서 자동 감지됩니다. 수동 지정이 필요하면 C:\NEIS\GOE\AtMessenger7\DB\@Talk.db 경로를 확인하세요.
백업 파일(.bak)은 어떻게 만드나요?
GOE 메신저 > 설정 > 백업에서 생성할 수 있습니다. 기본 저장 경로: C:\Users\사용자명\문서\GOE메신저\
다른 PC의 백업을 이 PC에서 볼 수 있나요?
네. .bak 파일을 복사해 온 후 "뷰어 DB 갱신"에서 추가하면 됩니다.
PIN을 잊었어요.
"사용자 재설정"으로 초기화한 후 다시 설정하세요. 원본 백업 파일(.bak)은 유지됩니다.
업데이트는 어떻게 하나요?
Microsoft Store에서 자동 업데이트됩니다. 새 버전이 있으면 프로그램 시작 시 안내가 표시됩니다.
데이터가 외부로 전송되나요?
대화 내용은 절대 외부로 전송되지 않습니다. 프로그램 시작 시 버전 확인과 익명 통계(OS, 메모리 등)만 전송됩니다. 자세한 내용은 이용 약관 4조를 참고하세요.
설정 직후 PIN을 또 입력하라고 합니다.
정상 동작입니다. 최초 설정 후 데이터 처리 전 PIN 확인을 한 번 더 요구합니다. 방금 설정한 PIN을 입력하면 됩니다.
